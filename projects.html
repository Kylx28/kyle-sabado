Formula 1 Data Visualization
This web app was made using Flask, HTML, CSS, and a python library called
FastF1 for the data. I enjoyed making this website because I recently
became a fan of Formula 1 and Ferrari in particular. This project taught
me how to analyze data using numpy and pandas, how to use Jupyter-Lab,
and how to make a web app using Flask.

Unfortunately, I was not able to deploy the Flask web app on a platform
like Heroku because the cache folder is too large. Without the folder, load
times would be much longer than it currently is.
